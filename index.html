Información y Manual de Operaciones

Detalles técnicos sobre el procesamiento de datos del Dashboard

Este sistema ha sido diseñado como un Dashboard Premium Tipo Power BI para el análisis exhaustivo de la matrícula, docentes, administrativos y locales educativos bajo la administración de la Dirección Regional de Educación de Piura (DREP).

1. Estructura de Datos Requerida (Excel)

Para cargar información actualizada, el archivo Excel debe contener 6 hojas obligatorias tituladas de cualquiera de las dos siguientes formas:

Hoja Nombre (Esquema A) Hoja Nombre (Esquema B) Columnas Requeridas
`TB_MATRICULA_GESTION_AREA` `TABLA 1: MATRICULA AREA Y GEOGRAFIA` Año, UGEL, Nivel, Gestión, Área, Total
`TB_MATRICULA_SEXO` `TABLA 2: MATRICULA SEXO` Año, UGEL, Nivel, Sexo, Total
`TB_DOCENTES` `TABLA 3: DOCENTES` Año, UGEL, Nivel, Gestión, Área, Total
`TB_NO_DOCENTE` `TABLA 4: NO DOCENTES` Año, UGEL, Nivel, Gestión, Área, Total
`TB_SERVICIOS` `TABLA 5: SERVICIOS EDUCATIVOS` Año, UGEL, Nivel, Gestión, Área, Total
`TB_LOCALES` `TABLA 6: LOCALES EDUCATIVOS` Año, UGEL, Nivel, Gestión, Área, Total

2. Seguridad y Persistencia de Actualizaciones

  • Cargar Excel (Consulta): Permite arrastrar o seleccionar un archivo de forma temporal. Al recargar la página, se perderán estos datos para no afectar a otros usuarios.
  • Actualizar Data (Administrador): Requiere la contraseña oficial (DREP2026). Al ingresar la clave correcta, los datos del Excel cargado se almacenan de forma persistente en el navegador (en localStorage). La base de datos permanecerá activa incluso si se apaga el equipo o se refresca el navegador.
  • Restablecer Demo: Si desea borrar los datos subidos y volver a ver los datos históricos de demostración, haga clic en el botón de restablecer data demo en la barra lateral.
